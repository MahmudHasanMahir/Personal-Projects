# Structural Frame Analysis

**Tools:** Space Gass

## Overview
Modelling and analysis of a steel portal frame structure using Space Gass, applying real-world load cases and verifying compliance with Australian structural design standards.

## Scope of Work
- Modelled a steel portal frame structure and applied dead, live and wind load cases.
- Generated member force diagrams: bending moment, shear force and axial force.
- Verified structural adequacy against Australian Standard AS 4100 (Steel Structures) requirements.
- Optimised member sizes to achieve an efficient, code-compliant design based on limit state design principles.
